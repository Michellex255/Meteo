**☁️ Meteo Weather App**

A dynamic weather application featuring real-time weather data and animated weather icons that respond to current conditions.

**Features**
- Live weather search: for any city worldwide
- Animated weather icons: clouds drift, sun pulses, rain falls
- 5-day weather forecast: with high/low temperatures
- Real-time data: including humidity, wind speed, and conditions
- Dynamic styling: that adapts to weather conditions (sunny, cloudy, rainy)
- Responsive design: for mobile and desktop

**Technologies Used**
- HTML5
- CSS3 (with custom animations)
- JavaScript (ES6)
- Axios for API calls
- SheCodes Weather API

**Live Demo**
[View Live Project](https://meteo-first-project.netlify.app/)

**How It Works**
Enter any city name in the search bar to get current weather conditions and a 5-day forecast. Watch the weather icons come alive with animations that match the current weather!
